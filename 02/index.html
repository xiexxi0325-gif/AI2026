창조의 근원, Self-Attention 아키텍처

Query (질의)

"나는 지금 어떤 정보를 찾고 있는가?"

모델이 현재 단어의 의미를 파악하기 위해 다른 단어들에게 던지는 질문입니다. 문맥의 핵심 질문을 형성합니다.

Key (색인)

"내 정보는 질문과 얼마나 관련 있는가?"

데이터베이스의 인덱스와 같은 역할을 합니다. Query와의 유사도를 측정하여 가중치를 결정합니다.

Value (값)

"내가 가진 실제 의미 정보는 무엇인가?"

Query와 Key의 유사도가 높을수록 이 Value 값이 최종 결과물 생성에 더 많이 반영됩니다.
